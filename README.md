# 알고리즘 스터디
백준 다이아를 목표로 매 주 주어지는 문제를 풀어서 올려봅시다.



# 방식
- 이 레포지토리를 clone해서 로컬 저장소를 만든다.
- 문제 풀이는 각자 본인 이름으로 만든 모듈 하위에 추가한다.
- 클래스 명에는 문제 번호를 작성한다.


# 문제 선정 및 풀이기간
- 매주 금요일에 issues탭에 각자 한 개의 문제를 올린다.
- 문제는 풀이를 완료하고 풀이한 내용을 PR을 올려 추가한다.
- 문제풀이에 대한 리뷰는 문제 출제일로부터 2주 이내에 완료한다.


# PR 형식
- 문제 풀이에 주요한 개념과 문제 풀이 방식을 작성한다.
- 코드 승인은 문제 풀이가 이해가 될 때 승인한다.
- PR merge 방식의 경우 Squash and merge 를 기본으로 한다.


그 외 어려운 점이나 모르는 부분은 슬랙 채널을 통해 소통한다.
